# HyperMem Memory Plugin

HyperMem Memory is the OpenClaw memory-slot plugin for HyperMem. It exposes HyperMem retrieval through OpenClaw's memory plugin surface for deployments that use the memory slot alongside, or instead of, the context-engine plugin.

## Install

```bash
npm install @psiclawops/hypermem-memory
```

See the main HyperMem install guide for runtime staging and OpenClaw activation:

- https://github.com/PsiClawOps/hypermem
- https://github.com/PsiClawOps/hypermem/blob/main/INSTALL.md
