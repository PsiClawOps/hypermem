# HyperMem 0.9.4 Release Notes

## Install

**npm:**
```bash
npm install @psiclawops/hypermem@0.9.4
```

**From source:**
```bash
git clone https://github.com/PsiClawOps/hypermem.git
cd hypermem && npm install && npm run build
```

See [INSTALL.md](../../INSTALL.md) for full setup instructions.

---

## Changes

- Recall warming is richer across bootstrap, warmup, and steady lifecycle bands.
- Smart recall widens for bootstrap/warmup, explicit `/new`, and topic-shift turns.
- Recent antecedents receive an adjacency-aware retrieval boost while heartbeat and system traffic stay filtered.
- Compose eviction protects the literal antecedent of the current user turn under non-critical pressure.
- `hypermem-install` stages recall-friendly defaults, doctor, bench, and package metadata from the published artifact.
- Existing HyperMem config is preserved during runtime staging.
- Installer checks now produce clear no-Ollama guidance and support explicit CI skip mode.
- Fresh-install smoke validation installs from the packed npm artifact without a source checkout.
- Failed install smoke runs preserve durable artifacts under `.artifacts/fresh-install-smoke/` for postmortem reuse.
